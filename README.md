# 🚗 配車作成ツール (haisha-scheduler)

集合場所や時間、運転者の空き状況に合わせて、乗客の配車を自動的に割り振るためのシンプルなWebツールです。

## ✨ 特徴

- **簡単なセットアップ**: 集合場所と時間を自由に追加・削除可能。
- **条件付き配車**: 運転者ごとに「集合場所」「時間」「乗車定員」を設定可能。
- **乗客の優先度**: 優先的に車に乗せる必要がある人を設定できる「優先」フラグ機能。
- **自動計算アルゴリズム**: 条件に一致する運転者と乗客をマッチング。定員を超えた場合は優先度に基づいて自動で振り分けます。
- **結果の共有**: 計算結果をワンクリックでテキストコピーして、LINEやメールですぐに共有できます。
- **ポータブル**: HTMLファイル1つで動作するため、サーバー不要でブラウザがあればどこでも使えます。

## 🚀 使い方

1. **設定**: 「集合場所」と「集合時間」を入力して追加します。
2. **運転者**: 名前、担当する場所・時間、乗せられる人数を入力して登録します。
3. **乗客**: 名前、希望する場所・時間を入力して登録します。特定のメンバーを優先させたい場合は「優先」を選択します。
4. **配車結果**: 「配車を計算する」ボタンを押すと結果が表示されます。「コピー」ボタンで結果をクリップボードに保存できます。

## 🛠 技術スタック

- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)

外部ライブラリやフレームワークは一切使用していないため、非常に軽量で高速に動作します。
